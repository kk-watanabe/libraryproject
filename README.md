## 学生時代のITスキル経験
大学ではFortranを用いてモンテカルロ法による核反応シミュレーションソフトの開発に取り組みました。物理・数学の知識をもとに計算モデルを構築し、数値計算の精度と処理効率の改善を意識して実装を行いました。特に、計算負荷の高い処理についてアルゴリズムを見直すことで、実行時間の短縮を図りました。

<br>

## 今後の展望
現在、就労移行支援事業所において、Djangoを用いた図書館管理アプリの開発に取り組んでいます。設計工程では、画面遷移図やER図、機能一覧の作成を行い、その後、基本的なCRUD機能の実装に加え、貸出・返却機能や予約機能の開発を進めてきました。また、ユーザー操作を意識した画面設計や、エラー処理、バリデーションの実装にも取り組み、実務を想定した開発を意識しています。今後はAPIの導入により、蔵書登録の自動化・効率化を図る予定です。開発は上司とスケジュールを共有し、進捗報告を行いながら進めており、実際の業務に近い形で取り組んでいます。
