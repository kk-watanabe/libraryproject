## 分散型図書館『テック図書館』
本アプリは、就労移行支援事業所から出された課題に取り組んで得た成果物です。開発は上司役のスタッフとスケジュールを共有し、進捗報告を行いながら進め、実際の業務に近い形で取り組みました。<br>

<br>

ビル内のあちこちに分散している数万冊の本をそれぞれの本棚に保管したまま一元管理する分散型図書館システムです。<br>
数万冊の本を一か所に集約する空きスペースがないことや、コスト・人手の面での維持管理が難しいといった課題に対するソリューションです。<br>
基本の貸出・返却・予約のみならず、ISBNバーコードスキャン&APIによる図書自動登録、レビュー機能を備え、ユーザー体験の向上を図りました。<br>

<br>

## 工夫した点
- 貸出は蔵書単位で管理 &rArr; 同じ本が複数ある場合に対応
- 保存場所のデータテーブル独立化 &rArr; 各本棚の本のフィルタリングを効率化
- バーコードスキャンによるISBN自動取得 &rArr; ISBN入力ミス防止
- APIを利用して書籍情報を自動取得 &rArr; 管理ユーザーの負担軽減
- 内部／外部レビュー機能 &rArr; 利用者体験を向上
- ボタンや状態表示の色分け &rArr; 視認性向上

## 学生時代のITスキル経験
大学ではFortranを用いてモンテカルロ法による核反応シミュレーションソフトの開発に取り組みました。物理・数学の知識をもとに計算モデルを構築し、数値計算の精度と処理効率の改善を意識して実装を行いました。特に、計算負荷の高い処理についてアルゴリズムを見直すことで、実行時間の短縮を図りました。<br>
また、既存の単粒子入射モデルを複合粒子入射モデルへ拡張する試みも行いました。拡張する際は既存のモデルをしっかり理解したうえで行うことを心がけ、拡張が自然になるようにしました。<br>
この時に培った、既存の仕組みをきちんと理解しそこに新しい機能を無理なく付け足していく考え方や、大量のデータを扱うときに、どうすればもっと速く、正確に処理できるかを突き詰める姿勢は、今のDjangoを使ったアプリ開発においても活きています。
